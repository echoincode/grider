# ETF项目扩展方案

## 一、策略扩展

### 1.1 多策略框架

| 属性 | 内容 |
|------|------|
| **功能描述** | 构建策略抽象层，支持多种交易策略（网格交易、双均线策略、MACD策略、布林带策略、动量策略等），替代单一网格策略。 |
| **价值原因** | 当前系统只支持网格交易一种策略，限制了用户的选择。多策略框架能满足不同市场环境下的交易需求，提升产品竞争力。 |
| **优先级** | ⭐⭐⭐ 高 |
| **难易程度** | 中 |
| **涉及地方** | `backend/app/algorithms/strategies/`（新建目录）、`backend/app/services/`、`frontend/src/features/` |

### 1.2 策略组合器

| 属性 | 内容 |
|------|------|
| **功能描述** | 支持策略叠加和切换，按权重融合多个策略信号，生成综合决策。 |
| **价值原因** | 单一策略在不同市场环境下表现差异大，策略组合能分散风险，提高整体收益稳定性。 |
| **优先级** | ⭐⭐ 中 |
| **难易程度** | 中 |
| **涉及地方** | `backend/app/algorithms/strategies/`、`backend/app/services/` |

---

## 二、投资组合管理

### 2.1 多标的投资组合

| 属性 | 内容 |
|------|------|
| **功能描述** | 支持多标的组合分析和风险分散，基于风险预算分配资金，优化标的权重。 |
| **价值原因** | 当前每次只分析单个ETF，无法实现资产配置和风险分散。多标的组合是专业投资者的核心需求。 |
| **优先级** | ⭐⭐⭐ 高 |
| **难易程度** | 高 |
| **涉及地方** | `backend/app/algorithms/portfolio/`（新建目录）、`backend/app/services/`、`backend/models/`、前端页面 |

### 2.2 资产配置工具

| 属性 | 内容 |
|------|------|
| **功能描述** | 根据用户风险偏好（保守/稳健/积极/激进）推荐资产配置比例（股票:债券:现金:商品）。 |
| **价值原因** | 帮助用户建立科学的资产配置方案，避免过度集中投资。 |
| **优先级** | ⭐⭐ 中 |
| **难易程度** | 中 |
| **涉及地方** | `backend/app/algorithms/portfolio/`、`backend/app/services/`、前端页面 |

---

## 三、分析工具增强

### 3.1 策略优化器

| 属性 | 内容 |
|------|------|
| **功能描述** | 自动寻找最优参数组合，支持遗传算法/网格搜索优化，优化目标为最大化夏普比率或最小化回撤。 |
| **价值原因** | 当前参数调整依赖经验判断，缺乏系统性的参数优化工具。策略优化器能显著提升策略表现。 |
| **优先级** | ⭐⭐ 中 |
| **难易程度** | 中 |
| **涉及地方** | `backend/app/algorithms/optimization/`（新建目录）、`backend/app/services/` |

### 3.2 因子分析模块

| 属性 | 内容 |
|------|------|
| **功能描述** | 分析影响ETF收益的因子（市场因子Beta、价值因子PB/PE、动量因子、质量因子ROE/ROA、波动率因子）。 |
| **价值原因** | 帮助用户理解ETF收益来源，识别风险暴露，做出更明智的投资决策。 |
| **优先级** | ⭐⭐ 中 |
| **难易程度** | 中 |
| **涉及地方** | `backend/app/algorithms/factors/`（新建目录）、`backend/app/services/`、前端展示 |

### 3.3 情景分析工具

| 属性 | 内容 |
|------|------|
| **功能描述** | 模拟不同市场情景下的策略表现（牛市+20%、熊市-20%、震荡市波动率翻倍、黑天鹅单日-10%）。 |
| **价值原因** | 帮助用户评估策略的鲁棒性，了解极端情况下的风险敞口。 |
| **优先级** | ⭐⭐ 中 |
| **难易程度** | 中 |
| **涉及地方** | `backend/app/algorithms/scenario/`（新建目录）、`backend/app/services/`、前端展示 |

## 四、数据可视化增强

### 4.1 高级图表组件

| 属性 | 内容 |
|------|------|
| **功能描述** | 新增多种高级图表：收益曲线对比图（策略收益 vs 基准指数）、回撤瀑布图、因子暴露热力图、资产配置饼图、策略参数敏感性图。 |

| **优先级** | ⭐⭐ 中 |
| **难易程度** | 中 |
| **涉及地方** | `frontend/src/features/analysis/components/`、图表库升级 |

### 4.2 综合仪表盘

| 属性 | 内容 |
|------|------|
| **功能描述** | 创建综合仪表盘页面，展示账户概览、收益曲线、持仓明细、市场热点、策略信号、风险提示等。 |
| **价值原因** | 提供一站式投资管理界面，提升用户体验和产品专业性。 |
| **优先级** | ⭐⭐ 中 |
| **难易程度** | 中 |
| **涉及地方** | `frontend/src/pages/Dashboard/`（新建目录）、`backend/app/routes/` |
